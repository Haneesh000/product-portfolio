Digital ATL Incharge
A product to automate work of the ATL In-Charges and Mentors.
List of Forms:-
  1. School form (To add new school)
  2. Student form (To add new student)
  3. Experiment form (To create new experiment)
  4. Team form (To create a new team)
  5. Community form (To add a new college or industry)
  6. Young mentors form (To add a new young mentor)
List of Reports:-
  • School report
  • Student report
  • Experiments report
  • Teams report
  • Communities report
  • Young mentors report
All the data entered in the forms will be shown in the reports. In reports we also get update and delete options
Features of the product:-
Time saver.
Saves a lot of time for the ATL In-Charges and mentors by automating most of their work. And also, it needs less human interaction.
Forms to collect data.
There are many forms which helps to collect data of students from mentors and In-Charges. Then this data is used for generating reports.
Automatic report generation.
In the present time to manually write reorts is a problem. It is solved by automatic report generation in the product.
Online
Can be accessed from any device which has a valid internet connection at any time and any place. If there is a report on paper we can easily lose it.
24 × 7 Availability
After the lab is closed then if the ATL In-Charge require report suddenly it is an issue for them. But, the product is 24 × 7 available so it is not an issue.
PWA Enabled
PWA (Progressive web app) allows you to install the site as an app inspite of being any operating system. So, you can use it as an app.
Technology Stack of the product:-
Google Firebase
Integrated with google firebase. A leading backend cloud service provider by google. Firebase is used in the product for database, authentication, etc.
NodeJS Powered
This product is powered with NodeJS a popular and powerful web framework for JavaScript. A lot of packages in npm are also used in the product.
Dart and Flutter powered
This product is also powered with Flutter a popular framework created by google in Dart programming language to create apps for any type of screens.
NPM powered
This product is also powered with NPM (Node Package Manager) a popular package manager for NodeJS.
EJS powered
This product also uses EJS (Embedded JavaScript Templating) a popular view engine which runs with NodeJS.
ExpressJS powered
This product also uses ExpressJS a popular backend URL routing framework engine which runs with NodeJS.
There are more npm packages used like ExpressJS, Lodash and EJS.
